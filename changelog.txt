# [2.0]
- Список доступных команд вынесен в отдельный файл;
- Команда !help теперь присылает справку по одной команде за сообщение. Сделано это из-за того, что общий размер справки привышает 2000 символов (сообщения не приходят моментально);
- Вся информация по командам берётся непосредственно с сервера, а не хранится в самом боте;
- Добавлена команда для создание игровых модулей. Команда !new_module зарегистрирует на ваше имя новый модуль;
- Добавлена команда для добавление игроков к модулю. !add_user позволит группировать нескольких пользователей за одним модулем;
- При критическом попадании в команде !attack бот пришлёт вам удивительное лицо;
- Более интерактивный подход к организации игры. Добавлена команда !invite с помощью которой можно пригласить игрока на указанный канал. Срок действия приглашения - 24 часа;
- Добавлена команда для просмотра созданных модулей. !show_modules отобразит список созданных вами модулей;
- Добавлена команда для просмотра данных о зарегистированных игроков. !show_users отобразит список игроков и модулей, в которых они учавствуют (предварительно необходимо использовать команду !add_user, чтобы игрок оказался в списке);
- В некоторых командах бот умеет читать текст в двойных кавычках. Он воспринимает такой текст как одно слово. Смотрите !help, чтобы узнать команды которые поддерживают данный функционал.
